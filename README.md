# TerrificEye3D: An Edge Computing System For Traffic Analytics From Videos

![title cover image](TerrificEye3D.gif)


## Requirements

* Python 3
* tensorflow
* DarkNet (Yolov4)
* ZED Camera (or Recorded 3D video files)
* GPU



## Running The 3D Analyzer

```
run.bat

```
